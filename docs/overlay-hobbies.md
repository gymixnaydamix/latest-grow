# Hobbies Overlay App

## Overview
Hobbies is the community overlay for crafts, photography, cooking, and interest-based learning.

## Metadata
- App ID: `hobbies`
- Label: `Hobbies`
- Category: `Community`
- Theme: `from-cyan-500 to-blue-600`
- Description: `Crafts, photography, and cooking communities`

## Primary Navigation
### Crafts
- `craft_tutorials`: Craft Tutorials
- `project_ideas`: Project Ideas
- `materials`: Materials
- `community`: Community

### Photography
- `photo_gallery`: Photo Gallery
- `photography_tips`: Photography Tips
- `camera_reviews`: Camera Reviews
- `photo_contests`: Photo Contests

### Cooking
- `recipe_library`: Recipe Library
- `cooking_classes`: Cooking Classes
- `ingredient_guide`: Ingredient Guide
- `meal_planning`: Meal Planning
- `community_competitions`: Community Competitions

## Default Entry
- Primary: `crafts`
- Secondary: `craft_tutorials`
