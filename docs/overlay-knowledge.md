# Knowledge Overlay App

## Overview
Knowledge is the learning overlay for books, courses, exams, certificates, and AI study assistance.

## Metadata
- App ID: `knowledge`
- Label: `Knowledge`
- Category: `Learning`
- Theme: `from-blue-500 to-indigo-600`
- Description: `Books, courses, exams, certificates, and AI study`

## Primary Navigation
### Books
- `digital_library`: Digital Library
- `ebooks`: E-Books
- `reading_lists`: Reading Lists
- `book_reviews`: Book Reviews

### Courses
- `course_catalog`: Course Catalog
- `my_courses`: My Courses
- `course_builder`: Course Builder
- `progress_tracking`: Progress Tracking

### Exams
- `exam_builder`: Exam Builder
- `test_scheduler`: Test Scheduler
- `grading_system`: Grading System
- `performance_reports`: Performance Reports

### Certificates
- `certificate_templates`: Certificate Templates
- `certificate_issuance`: Certificate Issuance
- `verification_system`: Verification System
- `digital_badges`: Digital Badges

### AI Study Assist
- `text_summarizer`: Text Summarizer
- `flashcard_generator`: Flashcard Generator
- `personalized_learning`: Personalized Learning
- `study_planner`: Study Planner

## Default Entry
- Primary: `books`
- Secondary: `digital_library`
