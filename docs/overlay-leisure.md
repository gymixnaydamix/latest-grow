# Leisure Overlay App

## Overview
Leisure is the community overlay for clubs, groups, calendars, and event booking flows.

## Metadata
- App ID: `leisure`
- Label: `Leisure`
- Category: `Community`
- Theme: `from-amber-500 to-orange-600`
- Description: `Community clubs and event bookings`

## Primary Navigation
### Clubs & Community
- `club_directory`: Club Directory
- `club_management`: Club Management
- `community_groups`: Community Groups
- `membership`: Membership

### Events Calendar & Booking
- `calendar_view`: Calendar View
- `event_creation`: Event Creation
- `booking_system`: Booking System
- `ticketing`: Ticketing

## Default Entry
- Primary: `clubs_community`
- Secondary: `club_directory`
