# Lifestyle Overlay App

## Overview
Lifestyle is the discovery and booking overlay for food, events, travel, flights, and cars.

## Metadata
- App ID: `lifestyle`
- Label: `Lifestyle`
- Category: `Lifestyle`
- Theme: `from-pink-500 to-fuchsia-600`
- Description: `Food, events, bookings, flights, and cars`

## Primary Navigation
### Food
- `restaurant_finder`: Restaurant Finder
- `reservations`: Reservations
- `delivery`: Delivery
- `reviews`: Reviews

### Events
- `event_finder`: Event Finder
- `ticket_booking`: Ticket Booking
- `event_calendar`: Event Calendar
- `my_events`: My Events

### Booking
- `hotel_booking`: Hotel Booking
- `property_rental`: Property Rental
- `vacation_packages`: Vacation Packages
- `travel_insurance`: Travel Insurance

### Flight
- `flight_search`: Flight Search
- `flight_booking`: Flight Booking
- `check_in`: Check-in
- `flight_status`: Flight Status

### Car
- `car_rental`: Car Rental
- `car_sharing`: Car Sharing
- `airport_transfers`: Airport Transfers
- `long_term_rental`: Long-term Rental

## Default Entry
- Primary: `food`
- Secondary: `restaurant_finder`
