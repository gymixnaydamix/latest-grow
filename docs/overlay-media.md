# Media Overlay App

## Overview
Media is the entertainment overlay for movies, series, documentaries, live TV, and kids content.

## Metadata
- App ID: `media`
- Label: `Media`
- Category: `Entertainment`
- Theme: `from-red-500 to-rose-600`
- Description: `Media viewing and content management hub`

## Primary Navigation
### Movies
- `movie_library`: Movie Library
- `categories`: Categories
- `watch_list`: Watch List
- `continue_watching`: Continue Watching

### Series
- `series_library`: Series Library
- `episodes`: Episodes
- `seasons`: Seasons
- `my_series`: My Series

### Docs
- `library`: Library
- `educational`: Educational
- `topics`: Topics
- `featured`: Featured

### Live TV
- No secondary navigation items

### Kids
- `kids_content`: Kids Content
- `educational`: Educational
- `entertainment`: Entertainment
- `parental_controls`: Parental Controls

## Default Entry
- Primary: `movies`
- Secondary: `movie_library`
