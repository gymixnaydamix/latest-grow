# Religion Overlay App

## Overview
Religion is the wellness overlay for texts, prayer tools, fasting support, and faith practice tracking.

## Metadata
- App ID: `religion`
- Label: `Religion`
- Category: `Wellness`
- Theme: `from-yellow-500 to-amber-600`
- Description: `Religious texts, practice tools, and trackers`
